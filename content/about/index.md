## 我的项目

以下是我目前正在维护和实验的一些主要项目：

* **OnyxAxis AI** (`ai.onyxaxis.org`)
    一站式大模型聚合平台，集成多款主流模型供免费体验。
    
* **Bouncer** (`bouncer-webui.pages.dev`)
    针对 Open WebUI 的一站式开源辅助工具，支持流控、公告、权限管理与商业化配置。
    
* **Veiled**
    隐身 AI 助手客户端，从安装到运行全程伪装为 Windows 系统显示驱动辅助工具。
    
* **PageDye**
    网页一键换肤扩展工具，兼容大部分主流网站，支持自定义更换网页壁纸。
    
* **ImmersiveProxy**
    将沉浸式翻译 API 逆向转换为 OpenAI 标准格式的轻量代理工具。
